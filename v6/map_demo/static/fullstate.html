Full-State Delta Sync

Same ticker data as the Stock Tickers demo, but delivered as a single delta-compressed frame per tick instead of individual per-key publications.

Centrifugo PRO only. The tick-based full-state channel (full_state_channel_prefix + full_state_tick_interval) is available in Centrifugo PRO.
How it works: The server serializes the entire map state into one JSON blob every 50ms (20 fps). Fossil delta compression computes the diff between consecutive frames. The client receives one small delta per tick instead of many individual key updates. Bandwidth savings scale with collection size.
