# Map Demo

Demonstrates Centrifugo map subscriptions with 9 interactive examples.

## Run

1. Start Centrifugo on port 8000: `centrifugo -c centrifugo.json`
2. Start demo: `docker compose up`
3. Open http://localhost:9000

## Prerequisites

- Centrifugo v6.7+ on port 8000
- centrifuge-js dev build on port 2000
- Docker Compose
